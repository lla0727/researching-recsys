# 社交媒体平台特性指南

## 微博 (Weibo)

### 平台特性
- **字数限制**：单条140字，可发多条
- **用户群体**：广泛，涵盖各年龄段
- **内容特点**：热点驱动、实时性强
- **互动方式**：转发、评论、点赞、@提及

### 文案风格要点
- 使用热点话题和标签
- 语言简洁有力，有爆点
- 适当使用表情符号增加活力
- 多用问句引导互动

### 热门标签类型
- 话题类：#热点事件 #生活日常
- 产品类：#好物分享 #使用测评
- 情感类：#治愈 #正能量
- 趣味类：#沙雕日常 #笑料

### 配图建议
- 尺寸：建议9张方形图或1张长图
- 风格：色彩鲜明、主题突出
- 要求：高清、有视觉冲击力

## 朋友圈 (Moments)

### 平台特性
- **字数限制**：无严格限制，建议300字内
- **用户群体**：熟人社交圈
- **内容特点**：真实生活分享
- **互动方式**：点赞、评论

### 文案风格要点
- 真实自然，像朋友聊天
- 分享个人真实体验
- 适当展现生活态度
- 避免过于商业化的表达

### 内容方向
- 生活记录：日常点滴、旅行见闻
- 情感表达：心情感悟、人生思考
- 美食分享：探店体验、自制美食
- 品牌推荐：真实使用感受

### 配图建议
- 尺寸：正方形或1:1比例
- 数量：1-9张，建议3-6张
- 风格：生活化、真实感
- 注意：图片质量要高，避免过度滤镜

## 小红书 (Little Red Book)

### 平台特性
- **字数限制**：标题20字，正文300-500字
- **用户群体**：年轻女性为主
- **内容特点**：种草、测评、分享
- **互动方式**：收藏、点赞、评论、关注

### 文案风格要点
- 标题吸引人，使用数字和表情
- 正文结构清晰，分段明确
- 大量使用emoji增加可读性
- 突出个人真实体验

### 经典结构
```
【吸睛标题】+ 数字/关键词/emoji

✨ 开场白（痛点/惊喜）
💡 产品介绍（核心卖点）
📝 使用感受（真实体验）
🎯 效果展示（对比/数据）
💰 价格信息（优惠信息）
🛒 购买建议（渠道/时机）

#标签1 #标签2 #标签3
```

### 热门标签
- #好物分享 #使用测评 #护肤美妆
- #生活好物 #家居好物 #美食探店
- #穿搭分享 #减肥瘦身 #运动健身

### 配图建议
- 尺寸：正方形，建议1080x1080
- 数量：3-9张，建议6-9张
- 风格：清晰、整洁、有质感
- 布局：首图要抓眼球，后续图有逻辑性

## 抖音 (TikTok)

### 平台特性
- **字数限制**：简洁明了，建议30字内
- **用户群体**：年轻用户为主
- **内容特点**：短视频、快节奏、强记忆点
- **互动方式**：点赞、评论、分享、关注

### 文案风格要点
- 节奏快，前3秒要抓住注意力
- 使用网络热词和流行语
- 配合视频内容，简洁有力
- 引导用户互动和行动

### 经典句式
- "家人们谁懂啊！"
- "XX天花板了！"
- "不允许你还不知道！"
- "最后3秒告诉你重点！"

### 话题挑战
- 参与#热门挑战
- 创造#原创话题
- 使用#BGM推荐
- @好友互动

### 配图建议
- 封面：高清、有冲击力
- 字幕：大字体、高对比度
- 标题：简洁、有悬念
- Logo：品牌标识明显

## 通用建议

### 内容创作原则
1. **真实性**：分享真实体验，避免虚假宣传
2. **价值性**：提供有用信息或情感共鸣
3. **原创性**：独特视角，避免千篇一律
4. **时效性**：结合最新热点和趋势

### 排雷指南
- 避免违规词汇和敏感内容
- 不攻击他人或品牌
- 不夸大产品功效
- 尊重平台规则和社区公约

### 发布最佳时间
- **工作日**：12:00-14:00，18:00-22:00
- **周末**：10:00-12:00，14:00-23:00
- **热点时刻**：重大事件发生时
