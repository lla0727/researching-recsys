
- 图像识别（需要额外OCR库）

## 建议的简单实现

### 1. 创建数据输入格式
```json
{
    "sampling_rate": 125,
    "data": [0.1, 0.2, -0.1, 0.3, ...],
    "metadata": {
        "brand": "oppo",
        "time": "2026-04-13",
        "heart_rate": 72
    }
}
```

### 2. 修改核心分析逻辑
只需要`ecg_processor.py`和`ecg_analyzer.py`，不需要PDF解析。

### 3. 使用方法
用户需要：
1. 从PDF中提取波形数据
2. 转换为JSON或CSV格式
3. 使用我们的分析工具
