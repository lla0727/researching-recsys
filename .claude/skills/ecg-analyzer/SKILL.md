# ECG 分析器

## 功能概述

本 Skill 用于分析ECG波形数据，提供专业的心电图分析报告。

**重要**: 本工具需要用户提供纯净的ECG数值数据，不支持直接解析PDF文件。

## 支持的功能

- **信号预处理**: 自动去噪、滤波，提高信号质量
- **心率分析**: 计算心率、检测心律不齐
- **早搏检测**: 识别房性早搏、室性早搏
- **ST段分析**: 检测ST段偏移、缺血可能
- **健康评估**: 综合分析结果，提供健康建议

## 数据格式要求

### 支持的输入格式

1. **JSON格式**
```json
{
    "sampling_rate": 125,
    "ecg_data": [0.1, 0.2, -0.1, 0.3, ...],
    "unit": "mV"
}
```

2. **CSV格式**
```csv
ecg_value
0.1
0.2
-0.1
0.3
...
```

3. **纯数值数组**
```
[0.1, 0.2, -0.1, 0.3, ...]
```

## 使用流程

1. **准备数据**
   - 从PDF中提取ECG波形数值（手动或使用工具）
   - 转换为JSON、CSV或数组格式

2. **提供数据**
   - 直接粘贴数值数组
   - 上传JSON/CSV文件
   - 或者逐行输入数据

3. **自动分析**
   - 信号预处理和增强
   - 计算各项指标
   - 生成分析报告

## 输出格式

### 分析报告结构

```
=== ECG 分析报告 ===
患者信息: [从PDF提取]
检测时间: [时间戳]

=== 基础指标 ===
- 平均心率: ___ bpm
- 心律: ___
- PR间期: ___ ms
- QT间期: ___ ms

=== 异常检测 ===
- 早搏: ___ (数量/类型)
- ST段: ___ (正常/偏移/异常)
- 其他异常: ___

=== 健康评估 ===
整体状况: [良好/一般/需要关注]

```

## 注意事项

- 本工具为辅助分析工具，不能替代专业医疗诊断
- 如有心脏不适，请及时就医
- 分析结果仅供参考，请结合临床症状判断

## 技术实现

- PDF解析: 使用PyPDF2/pdfplumber提取数据
- 信号处理: 使用scipy进行滤波和去噪
- 算法: 基于心电图分析标准实现
