# 从Oppo ECG PDF中提取数据的方法

## 方法1：手动提取（推荐）

如果你的Oppo PDF是**文字+波形图**混合格式：

1. **用Adobe Acrobat打开PDF**
2. **选择文本工具**
3. **尝试选中波形数据**（如果能选中）
4. **复制粘贴到文本编辑器**
5. **提取数值**，格式化为CSV或JSON

## 方法2：截图提取

1. **将波形图截图保存**
2. **使用图像转文本工具**（如OCR）
3. **识别出数值点**

## 方法3：数据清理模板

如果PDF中有文字格式的数据，可以参考以下模式提取：

```
示例数据格式：
时间: 10:30:25
心率: 72 bpm
数值: 0.15, 0.20, -0.10, 0.25, -0.05, 0.30, ...
```

## 提取后的数据格式

### JSON格式（推荐）
```json
{
    "brand": "oppo",
    "sampling_rate": 125,
    "duration": 10,
    "ecg_data": [
        0.15, 0.20, -0.10, 0.25, 
        -0.05, 0.30, 0.10, -0.20,
        0.35, -0.15, 0.05, 0.40,
        ...
    ]
}
```

### CSV格式
```csv
ecg_value
0.15
0.20
-0.10
0.25
-0.05
0.30
...
```

## 注意事项

- 确保数据是连续的ECG波形值
- 单位通常是毫伏(mV)
- 采样率通常是125Hz（Oppo常见）
- 至少需要几秒的数据（建议10秒以上）

## 提示

如果PDF中的数值是**以图表形式存在**而不是纯文本，你可能需要：
1. 使用坐标拾取工具获取波形点
2. 或者使用专业数据提取软件

提取完成后，就可以使用我们的ECG分析工具了！
